# ChampSim – UCP Cache Partitioning Project

This repository is a fork of the official ChampSim simulator and is used to implement and evaluate Utility-Based Cache Partitioning (UCP).

Project phases:
1. Baseline Shared LLC with LRU
2. Static Way Partitioning
3. Utility-Based Cache Partitioning (UCP)

Original project: https://github.com/ChampSim/ChampSim
